TITRE: La propriété background en CSS
DESCRIPTION: Tout sur les arrière-plans en CSS
NIVEAU: Débutant

## DÉFINITION
La propriété CSS `background` est une propriété raccourcie qui permet de définir toutes les propriétés d'arrière-plan d'un élément en une seule déclaration. Elle contrôle l'apparence visuelle de l'arrière-plan d'un élément HTML.

## RÔLE
Le rôle principal de la propriété background est de :
- Définir une couleur d'arrière-plan unie
- Ajouter une ou plusieurs images d'arrière-plan
- Contrôler le positionnement des images
- Définir le comportement de répétition
- Spécifier la taille de l'image
- Créer des dégradés de couleur

## USAGE
La propriété background s'utilise sur n'importe quel élément HTML. Voici les bonnes pratiques :
1. Utilisez des couleurs avec un bon contraste pour la lisibilité
2. Optimisez les images pour le web
3. Utilisez des dégradés pour des effets modernes
4. Testez sur différents appareils et tailles d'écran

## EXEMPLE 1 - Couleur unie
CODE: 
.boite {
    background: #3498db;
    width: 200px;
    height: 200px;
    border-radius: 10px;
}

RÉSULTAT: <div style="background: #3498db; width: 200px; height: 200px; border-radius: 10px; display: flex; align-items: center; justify-content: center; color: white; font-weight: bold;">Couleur unie</div>

## EXEMPLE 2 - Image d'arrière-plan
CODE:
.hero {
    background: url('image.jpg') center/cover no-repeat;
    height: 300px;
    position: relative;
}

RÉSULTAT: <div style="background: linear-gradient(45deg, #3498db, #2ecc71); height: 150px; display: flex; align-items: center; justify-content: center; border-radius: 10px; color: white; font-weight: bold;">Image ou dégradé simulé</div>

## EXEMPLE 3 - Dégradé linéaire
CODE:
.bouton {
    background: linear-gradient(to right, #ff7e5f, #feb47b);
    padding: 12px 24px;
    border: none;
    border-radius: 25px;
    color: white;
    cursor: pointer;
}

RÉSULTAT: <button style="background: linear-gradient(to right, #ff7e5f, #feb47b); padding: 12px 24px; border: none; border-radius: 25px; color: white; cursor: pointer; font-weight: bold;">Bouton avec dégradé</button>

## EXEMPLE 4 - Plusieurs arrière-plans
CODE:
.card {
    background: 
        linear-gradient(rgba(0,0,0,0.5), rgba(0,0,0,0.5)),
        url('texture.jpg');
    background-size: cover;
    padding: 20px;
    color: white;
}

RÉSULTAT: <div style="background: linear-gradient(rgba(0,0,0,0.7), rgba(0,0,0,0.7)), linear-gradient(135deg, #667eea, #764ba2); padding: 20px; border-radius: 10px; color: white; height: 100px; display: flex; align-items: center; justify-content: center; font-weight: bold;">Arrière-plan superposé</div>

## PROPRIÉTÉS DÉTAILLÉES
- `background-color` : Définit la couleur d'arrière-plan
- `background-image` : Définit une ou plusieurs images
- `background-position` : Position de l'image (center, top, left, etc.)
- `background-size` : Taille de l'image (cover, contain, valeurs)
- `background-repeat` : Répétition (no-repeat, repeat-x, repeat-y)
- `background-attachment` : Fixe ou défile avec le contenu

## ASTUCES
1. Utilisez `background-clip: text` avec `-webkit-text-fill-color: transparent` pour du texte avec dégradé
2. Les dégradés CSS remplacent souvent les images et améliorent les performances
3. Utilisez des couleurs semi-transparentes pour superposer du contenu
4. Testez toujours le contraste texte/arrière-plan pour l'accessibilité

## SUPPORT NAVIGATEURS
La propriété background est supportée par tous les navigateurs modernes. Pour les dégradés, vérifiez les préfixes pour les anciennes versions.
